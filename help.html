Keyboard Shortcuts

Ctrl + O — Open files

Ctrl + S — Save work

Ctrl + N — New document

Ctrl + F — Focus search

Ctrl + H / F1 — Help

Esc — Close modal

Supported Formats

  • Images: .jpg, .jpeg, .png, .gif, .bmp, .webp

Frequently Asked Questions

Processing local hota hai?

Haan, tamam processing aapke browser me local hoti hai. Aapki files device se bahar nahi jati.

Supported file types & size?

PDF files up to 50MB support hain. Images: JPG, PNG, GIF, BMP, TIFF.

Privacy & security?

Local processing ki wajah se data secure rehta hai. Sensitive files ko upload karne ki zarurat nahi.

Common errors kaise solve karein?

Agar file corrupt ya bohot badi ho to split/compress try karein. Browser memory limit par heavy PDFs fail ho sakti hain.
