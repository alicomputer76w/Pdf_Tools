About PDF Tools Suite

Fast, privacy‑focused PDF utilities that run entirely in your browser.

Mission

PDF Tools Suite aims to make everyday document tasks simple, secure, and accessible — without uploading your files. All processing happens locally in your browser using modern Web APIs and libraries.

Key Features

  • Merge, split, rotate, compress and optimize PDFs
  • Convert between PDF and images (PNG/JPEG), and export text
  • Add watermarks, page numbers, bookmarks, annotations, and signatures
  • Batch processing and drag‑and‑drop support
  • Works offline with Service Worker caching

Privacy by Design

Your files never leave your device. We do not transmit, store, or analyze file contents on any server. Temporary data stays in memory and is cleared when you refresh or close the tab.

Technology

  • pdf-lib for PDF manipulation
  • pdf.js for rendering and text extraction
  • Modern JavaScript (ES6+), Web Workers, and Canvas APIs
  • Progressive Web App features for offline use

Accessibility

We follow accessible UI patterns and provide keyboard navigation, focus management, and high‑contrast modes.
